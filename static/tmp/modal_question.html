How to use IT shpora

First steps

1)First of all you need create an account for your personal instance of code notes.
If no – you can see my code notes but without any CRUD permission.

2)To be able to add first CODE NOTE – you need add at least one SECTION +add section .

Conventional cases:

  • Use QUICK ADD to quickly add new code note which you may need remember in Future. By default it is the first tab to load if you signed.
  • Use SEARCH engine to quickly find your prompts.

CRUD SECTION

All sections sorted by name.

  • You can ADD new sections as admitted above
  • You can RENAME section
  • You can REMOVE section. ALL DOCS FROM THIS SECTION WILL BE DELETED
  • You can MERGE section. It means dispatching all code notes from these section to picked.

CRUD code notes

All code notes sorted by created date. If you dispatch code note to another section – date will renewed.

  • Use QUICK ADD tab and quickly add code note
  • Use +NOTES of particular section to add new code note in these section.
  • You can REMOVE code note
  • You can DISPATCH code note to another section
  • You can EDIT code note

Search

USE search engine to quickly find code notes like put elastic

Technologies I have used:

  • Elasticsearch (as main storage and search engine)
  • MongoDB (for users)
  • Python (flask)
  • AngularJS (front-end framework)
